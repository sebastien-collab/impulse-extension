## Full Description

### Media Buyer Helper — Universal Pixel Debugger & Tech Stack Detector

Media Buyer Helper instantly detects tracking pixels and technologies running on any webpage. Built for digital marketers who need to verify their tracking implementations quickly.

**What it does:**

The extension monitors webpages in real time using three detection methods — JavaScript interception, DOM scanning, and network monitoring — to identify which tracking platforms are active and extract their configuration details.

**5 built-in tabs:**

**Pixels** — See every active tracking pixel on the current page. View pixel IDs, event timelines with full arguments, and detection source for each platform.

**Consent** — Check Google Consent Mode V2 compliance. Verify default/update commands, detect missing parameters, and identify Basic vs Advanced mode.

**Ad Libraries** — Quick links to major ad transparency tools for the current domain. Research any advertiser's active campaigns in one click.

**Stack** — Detect the technologies powering any website — CMS, JavaScript frameworks, CSS frameworks, CDN, analytics, and marketing tools.

**Tools** — Built-in utilities: color picker with hex/RGB/HSL values and history, font inspector to identify typefaces on hover, and one-click cookie/cache clearing for the current site.

**Key features:**
- Bilingual interface (English / French)
- Light and dark theme
- Lightweight with zero external requests
- Everything processed locally in your browser

**Privacy:**
Media Buyer Helper processes everything locally. No data is collected, stored, or sent anywhere. Open source on GitHub.

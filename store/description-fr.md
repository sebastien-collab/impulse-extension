### Media Buyer Helper — Debugger Universel de Pixels & Detecteur de Tech Stack

Media Buyer Helper detecte instantanement les pixels de tracking et les technologies presentes sur n'importe quelle page web. Concu pour les marketeurs digitaux qui ont besoin de verifier leurs implementations de tracking rapidement.

**Ce que ca fait :**

L'extension surveille les pages web en temps reel via trois methodes de detection — interception JavaScript, scan DOM et monitoring reseau — pour identifier les plateformes de tracking actives et extraire leurs details de configuration.

**5 onglets integres :**

**Pixels** — Visualisez chaque pixel de tracking actif sur la page courante. Consultez les Pixel ID, les timelines d'evenements avec arguments complets, et la source de detection pour chaque plateforme.

**Consent** — Verifiez la conformite Google Consent Mode V2. Controlez les commandes default/update, detectez les parametres manquants, et identifiez le mode Basic vs Advanced.

**Ad Libraries** — Liens rapides vers les principaux outils de transparence publicitaire pour le domaine courant. Recherchez les campagnes actives de n'importe quel annonceur en un clic.

**Stack** — Detectez les technologies qui font tourner n'importe quel site web — CMS, frameworks JavaScript, frameworks CSS, CDN, analytics et outils marketing.

**Tools** — Utilitaires integres : pipette couleur avec valeurs hex/RGB/HSL et historique, inspecteur de polices pour identifier les typos au survol, et suppression des cookies/cache du site courant en un clic.

**Fonctionnalites cles :**
- Interface bilingue (Francais / English)
- Theme clair et sombre
- Leger, aucune requete externe
- Tout est traite localement dans votre navigateur

**Vie privee :**
Media Buyer Helper traite tout localement. Aucune donnee n'est collectee, stockee ou envoyee. Open source sur GitHub.
